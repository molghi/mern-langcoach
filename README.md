# Full-Stack: LangCoach

### Vocabulary Trainer app

A minimalist language trainer app that allows users to:

- Create personal accounts
- Add new words or phrases
- View and manage all entries in one place (edit, delete)
- Filter entries by language, category or creation date
- Practice by language using quiz mode with spaced repetition
- Import and export words data

**Tech stack:** MERN, React/TypeScript, Tailwind.
